# Event library for mobile
MonoEvent 是专门用于处理移动端的事件多投绑定、卸载、代理，支持诸如 tap、doubleTap、longTap、swipe(Left/Right/Up/Down) 等常用的手势事件，后续会根据情况考虑加入更多的特殊事件类型。
